# 🎙 Enterprise Meeting Copilot

A voice-powered AI copilot that helps Enterprise Account Executives prepare for executive meetings.

Built with:

- 🎙 ElevenLabs
- 🐍 Python
- ⚡ Flask
- ☁️ Render

---

## Why I built it

As an Enterprise Account Executive, I spend a significant amount of time preparing for executive meetings.

I wanted to explore how voice AI could make that workflow faster by turning meeting preparation into a short audio briefing that can be listened to while commuting or between customer meetings.

---

## Current features

- Generate a structured executive meeting briefing
- Voice-powered briefing using ElevenLabs
- Executive meeting template
- Enterprise-focused discovery questions
- Suggested meeting opener
- Objection preparation
